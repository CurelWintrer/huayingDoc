# 靠泊

## 一、靠泊页面

![靠泊页面](./img/berth/靠泊页面.png)

1. 选择泊位按钮
2. 回到船舶当前位置按钮
3. 实时轨迹显示开关
4. 当前速度显示
5. 推进器转速
6. 推进器角度



## 二、靠泊点编辑

首先点击“选择泊位按钮”

![选择泊位按钮](./img/berth/选择泊位按钮.png)



地图中心会自动回到船舶位置，并放大到适合选择泊位的缩放等级，然后显示一个矩形用于表示泊位，该矩形可以任意拖动和旋转。

![泊位编辑界面](./img/berth/泊位编辑界面.png)

1. 泊位大小和位置矩形
2. 确定泊位按钮
3. 泊位旋转按钮
4. 泊位保存按钮
