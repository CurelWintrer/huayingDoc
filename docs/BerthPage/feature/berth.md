# 靠泊

## 一、靠泊页面

靠泊页面用于辅助船舶进行泊位对准与靠泊操作，提供实时航行状态与推进器控制信息。

![靠泊页面](./img/berth/靠泊页面.png)

1. **选择泊位按钮：** 进入泊位选择与编辑模式，用于设定目标泊位区域
2. **回到船舶当前位置按钮：**将地图视角快速定位至船舶当前位置
3. **实时轨迹显示开关：**控制是否显示船舶实时运行轨迹
4. **当前速度显示：**显示船舶当前航行速度
5. **推进器转速：**显示当前推进器的转速状态
6. **推进器角度：**显示推进器当前角度信息



## 二、靠泊点编辑

首先点击“选择泊位按钮”

![选择泊位按钮](./img/berth/选择泊位按钮.png)



系统将自动执行以下操作：

- 地图中心自动回到船舶当前位置
- 地图缩放至适合泊位选择的视图等级
- 显示可编辑泊位矩形区域

泊位以**可拖动、可旋转的矩形区域**形式呈现，用于精确设定靠泊位置。

![泊位编辑界面](./img/berth/泊位编辑界面.png)

1. **泊位区域矩形：**用于表示当前选定的泊位范围，可进行位置调整
2. **确定泊位按钮：**确认当前泊位设置并进入保存流程
3. **泊位旋转控制：**用于调整泊位朝向角度，使其与实际码头方向一致
4. **泊位保存按钮：**保存当前泊位配置，供后续快速调用
