# 泊位保存

## 一、保存泊位

泊位保存功能用于将已编辑的泊位区域保存至本地，便于后续快速调用与重复使用。在泊位编辑模式下，右侧工具栏提供“保存泊位”按钮。

![泊位保存按钮](./img/saveBerthPosition/泊位保存按钮.png)

点击该按钮后，将弹出“保存泊位”确认窗口，用户需输入泊位名称。确认信息无误后，点击“确定”即可完成保存，泊位将存储至本地。

![泊位保存弹窗](./img/saveBerthPosition/泊位保存弹窗.jpg)

## 二、加载已保存泊位

已保存的泊位可通过“我的泊位”列表进行加载。在靠泊页面中，长按“靠泊”按钮，即可打开“我的泊位”弹窗。

在弹窗中选择目标泊位后，点击“确定”，系统将自动将该泊位加载至地图中并生效

![保存的泊位弹窗列表](./img/saveBerthPosition/保存的泊位弹窗列表.jpg)
