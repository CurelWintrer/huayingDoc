天气信息卡片位于主页左上区域，用于显示船舶当前位置的实时天气状况，为船员提供航行和作业环境参考。

#### 显示内容

天气信息区域包含以下内容：

##### 1、当前温度

显示当前位置实时气温。

单位：

```
℃
（摄氏度）
```

例如：

```
23.2℃
```

该温度数据来源于天气服务平台，仅供参考。

##### 2、当前地址

显示船舶当前位置对应的地理位置信息。

##### 3、天气情况

显示当前天气状态。

##### 4、天气图标

使用图形化方式显示当前天气状态。



#### 重新加载天气数据

天气信息依赖网络连接获取。

在以下情况下可能导致天气数据加载失败：

- 网络连接异常
- 移动网络信号较弱
- 卫星网络通信延迟
- 天气服务暂时不可用

当天气数据获取失败时，界面会显示加载失败提示。

此时可点击：

```
重新加载
```

按钮重新请求天气数据。

系统将重新获取当前位置对应的天气信息。
