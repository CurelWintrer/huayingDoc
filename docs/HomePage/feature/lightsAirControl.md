# 灯光-空调控制



## 一、灯关控制

![灯光控制](./img/lightsAirControl/灯光控制.png)

#### 1. 所有灯光控制开关

用于统一控制船舶全部灯光设备。

开启后：

- 导航灯
- 锚灯
- 顶灯
- 地灯

将根据当前配置统一开启。

关闭后：

- 所有受控灯光将统一关闭。

该功能适用于快速开启或关闭整船照明系统。



#### 2. 导航灯控制开关

用于控制船舶导航灯。

#### 3. 锚灯控制开关

用于控制船舶锚灯。

#### 4. 顶灯控制开关

用于控制船体顶部照明灯。



#### 5. 地灯控制开关

用于控制船体地灯或甲板辅助照明灯。



#### 开关状态说明

灯光控制采用滑动开关方式进行操作。

| 开关状态 | 说明           |
| -------- | -------------- |
| 开启     | 对应灯光已打开 |
| 关闭     | 对应灯光已关闭 |

点击或滑动开关即可切换灯光状态。



#### 状态同步机制

灯光控制页面显示的是船体灯光的实际工作状态。

当出现以下情况时，界面状态将自动同步更新：

- 用户在当前页面操作灯光开关
- 用户在智能设备控制页面操作灯光
- 船载控制系统直接控制灯光
- 灯光设备状态发生变化

因此，界面显示状态始终与船体实际灯光状态保持一致。



#### 智能设备控制页面

除主页快捷控制外，系统还在《[智能设备控制页](../../ShipPage/shipPage.md)》中提供完整的灯光管理功能。

智能设备控制页面可显示：

- 每个灯光设备的名称
- 灯光实时状态
- 灯光安装位置
- 灯光控制功能

便于船员快速识别和管理船体各位置灯光设备。

#### 注意事项

- 夜间航行时应按照当地航行规则正确使用导航灯。
- 锚泊期间应根据实际情况开启锚灯。
- 长时间开启高功率照明设备可能增加电池能耗。
- 若开关状态与实际灯光状态不一致，请检查灯光设备和通信状态是否正常。





## 二、空调控制

![空调控制](./img/lightsAirControl/空调控制.png)

空调控制区域位于主页右下区域，用于显示和控制船舱空调设备运行状态。

船员可通过该区域快速查看当前空调工作模式、设定温度及风速等级，并进行快捷调节，无需进入智能设备控制页面即可完成常用操作。



#### 1. 空调开关

用于控制空调设备的启动和关闭。

状态说明：

| 状态 | 说明           |
| ---- | -------------- |
| 开启 | 空调正在运行   |
| 关闭 | 空调已停止运行 |

开启后，空调将按照当前设定的模式、温度和风速运行。

#### 2. 空调模式

显示当前空调运行模式。

支持以下模式：

| 模式 | 说明         |
| ---- | ------------ |
| 制冷 | 降低舱内温度 |
| 制热 | 提高舱内温度 |
| 除湿 | 降低空气湿度 |

#### 3. 空调温度

显示当前设定温度。

#### 4. 空调风速

显示当前风速等级。

支持以下档位：

| 风速 | 说明                     |
| ---- | ------------------------ |
| 低速 | 风量较小，运行噪声较低   |
| 中速 | 适用于日常使用           |
| 高速 | 快速调节舱内温度         |
| 自动 | 根据环境温度自动调整风速 |

#### 5. 最低支持温度

显示空调允许设置的最低温度。

#### 6. 空调模式切换按钮

用于切换空调运行模式。

点击后可循环切换：

```
制冷 → 制热 → 除湿 → 制冷
```

#### 7. 空调风速切换按钮

用于切换风速等级。

点击后可循环切换：

```
低速 → 中速 → 高速 → 自动
```

#### 8. 降低温度按钮（－）

用于降低空调设定温度。

每点击一次：

```
温度降低 1℃
```

#### 9. 升高温度按钮（＋）

用于提高空调设定温度。

每点击一次：

```
温度升高 1℃
```

#### 10. 最高支持温度

显示空调允许设置的最高温度。





#### 注意事项

- 空调关闭状态下无法调整温度和风速。
- 制冷模式下温度设置过低会增加电池能耗。
- 长时间使用高速风速可能增加设备功耗。
- 自动模式下风速由空调设备自动控制。
- 若空调状态长时间未更新，请检查设备连接状态是否正常。
