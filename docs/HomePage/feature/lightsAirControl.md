# 灯光-空调控制



## 一、灯关控制

![灯光控制](./img/lightsAirControl/灯光控制.png)

1. 所有灯光控制开关
2. 导航灯控制开关
3. 锚灯控制开关
4. 顶灯控制开关
5. 地灯控制开关

滑动开关可以打开和关闭灯光，并且开关的状态也会根据船体的实际灯光改变。[智能设备控制页](../../ShipPage/shipPage.md)同样提供了灯光控制功能，并且显示了每个灯在船体的位置信息。
