# 电子围栏

围栏保证船舶在可控区域内安全航行，App还提供了电子围栏功能，您可以在地图上框选区域设定电子围栏。您本次设定的电子围栏在您关闭App后不会删除，下次打开时默认使用您设定好的电子围栏。

当船舶不慎驶出围栏区域后，平板会发出声光报警，提醒您驶出了围栏区域。



## 一、点击右下角电子围栏绘制按钮，进入电子围栏绘制界面

![电子围栏按钮](./img/electronicFence/电子围栏按钮.png)



## 二、绘制电子围栏

电子围栏的显示效果是一个闭合半透明多边形，绘制方式和路径点绘制方式类似。

![电子围栏绘制](./img/electronicFence/电子围栏绘制.png)

1. 围栏节点
2. 线段中点，点击或拖动可以在中点位置插入围栏节点
3. 撤销按钮
4. 重做按钮
5. 删除编辑的电子围栏
6. 节点详情，点击右侧删除按钮可以删除当前点
7. “设置围栏”按钮，点击后显示确认弹窗
8. “取消围栏”按钮，点击后将取消已经设定好的电子围栏
9. “退出编辑”按钮，点击后退出编辑模式，隐藏电子围栏编辑工具



## 三、设置电子围栏

电子围栏绘制完成后，即可点击“设置围栏”按钮，然后点击弹窗的确认键，电子围栏设置成功。

![电子围栏设置弹窗](./img/electronicFence/电子围栏设置弹窗.jpg)
