# 航线保存

App提供了航线保存功能，您可以将绘制好的航线（包括预设的速度）保存到本地，下次打开可以直接选择已经保存的航线进行航行，这在需要经常使用复杂航线的场景下非常适用，这将为您节省大量宝贵的时间。



## 一、保存航线

对于已经绘制好的航线，您可以点击绘制工具栏右侧的保存按钮保存航线

![保存航线按钮](./img/wayPointSave/保存航线按钮.png)

点击后将弹出一个保存确认弹窗，您需要输入一个航线名称，虽然App为您提供了一个默认名称（默认为当前时间），但是我们还是建议您输入一个具有代表性的名称，避免遗忘。

![保存航线弹窗](./img/wayPointSave/保存航线弹窗.jpg)

最后点击弹窗的“保存”按钮，任务即保存到本地。

## 二、航线加载

加载保存好的航线同样很简单，您只需点击右下角的加载按钮，即可弹出航线列表

![本地任务加载按钮](./img/wayPointSave/本地任务加载按钮.png)

在“本地任务”弹窗中，能看到前面保存好的“滴水湖1号线”

![航线列表弹窗](./img/wayPointSave/航线列表弹窗.jpg)

点击该任务后再点击确认按钮，航线便能加载到地图上：

![加载的航线](./img/wayPointSave/加载的航线.jpg)
