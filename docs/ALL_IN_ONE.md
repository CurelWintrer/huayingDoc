## 请在使用前仔细阅读

本手册适用于智能船舶控制平板应用程序（以下简称“本系统”）。在使用本系统前，请仔细阅读并理解本手册中的全部内容，以确保安全、正确地使用各项功能。



## 重要安全提示

本系统为船舶辅助控制与监测系统，可用于船舶状态监控、航线规划、自动循迹、电子围栏管理等功能，但不能替代船员的专业判断和现场操作。

在使用过程中，请遵守以下安全要求：

### 保持人工监控

无论船舶处于手动驾驶还是自动循迹状态，船员均应保持对周围环境、航道情况及船舶状态的持续监控。

### 自动功能不等于无人驾驶

自动循迹功能仅根据预设航线控制船舶航行，不具备完全自主避碰能力。船员应随时做好接管准备，并根据实际情况进行人工干预。



### 紧急情况下立即接管

当出现以下情况时，应立即停止自动控制并切换至人工驾驶：

- 船舶偏离预定航线
- 周边出现障碍物或其他船舶
- 恶劣天气影响航行安全
- GNSS定位异常
- 通讯链路异常
- 控制设备报警

### 注意定位精度限制

本系统依赖GNSS卫星定位数据。受天气、环境遮挡、电磁干扰等因素影响，定位精度可能出现波动。

在以下环境中应谨慎使用自动控制功能：

- 桥梁下方
- 港区密集建筑区域
- 山体遮挡区域
- 恶劣天气环境

### 注意设备状态

航行前应确认：

- 平板设备电量充足
- 控制器工作正常
- GNSS状态正常
- 网络连接正常
- 推进系统工作正常

### 数据仅供参考

本系统显示的船位、速度、航向、航迹等信息来源于船载设备和传感器数据，仅供辅助决策参考。船员应结合现场实际情况进行判断。

## 免责声明

因设备故障、通讯异常、卫星信号中断、第三方地图数据误差、不可抗力因素或不当操作导致的航行风险和损失，系统开发方不承担由此产生的直接或间接责任。

用户应遵守当地法律法规及相关航行规定，并对船舶运行安全承担最终责任。

# 二、系统简介

软件版本：1.0.0

![平板展示](./img/systemIntroduction/平板展示.png)

## 系统概述

该系统是一套集航线规划、自动循迹、电子围栏和设备管理于一体的智能化船舶辅助控制平台。

船员可通过平板应用实时查看船舶运行状态，并完成航行任务管理和自动控制操作，提高航行效率和作业安全性。

## 主要功能

### 实时监控

实时显示：

- 船舶位置
- 船舶轨迹
- 航向信息
- 航行速度
- 推进器状态
- 电源状态
- 通讯状态

### 航线管理

支持：

- 创建航线
- 编辑航线
- 保存航线
- 删除航线

### 自动循迹

根据预设航线自动控制船舶航行。

### 电子围栏

支持创建和管理电子围栏区域，实现：

- 离开区域提醒
- 越界报警

### 历史记录

支持查看：

- 消息历史
- 报警记录

### 报警管理

当设备或船舶状态异常时，系统能够及时发出报警提示，帮助船员快速发现并处理问题。

# 四、航行状态显示

## 一、航速仪表盘

![组合 13](./HomePage/feature/img/sailingState/速度仪表盘.png)



航速仪表盘位于主页左下区域，用于实时显示当前船舶航行速度，帮助船员快速掌握船舶运行状态。

#### 功能说明

仪表盘以指针表盘形式显示船舶当前航速，数据来源于船载定位系统（GNSS）和船舶控制系统。

显示内容包括：

- 当前航速数值
- 航速单位（m/s）
- 仪表盘量程范围
- 当前速度对应的刻度位置

系统会根据实时数据自动刷新显示内容。

#### 航速单位

航速采用国际单位制：

```
m/s（米每秒）
```



## 二、推进器信息



![组合 14](./HomePage/feature/img/sailingState/推进器信息.png)

推进器信息卡片位于主页中部区域，用于实时显示船舶推进器的运行状态，包括推进器转速、转向角度、推力方向及工作状态等信息。

通过该区域，船员可快速了解当前推进系统运行情况和船舶动力输出状态。



#### 1. 推进器转速（Thruster RPM）

显示当前推进器实际转速。

单位：

```
RPM（Revolutions Per Minute，转/分钟）
```

转速数值反映推进器当前输出功率。

一般情况下：

- 转速越高，推进器输出推力越大
- 转速越低，推进器输出推力越小
- 转速为 0 RPM 时，推进器处于停止状态

船员可通过转速变化判断船舶当前动力输出情况。

#### 2. 推进器角度数值（Angle）

显示推进器当前转向角度。

单位：

```
DEG（Degree，度）
```

角度数值表示推进器相对于船体中心线的偏转方向。

例如：

| 角度 | 说明           |
| ---- | -------------- |
| 0°   | 推进器朝向船首 |
| 30°  | 向右偏转30°    |
| -30° | 向左偏转30°    |
| 90°  | 侧向推进       |

推进器角度直接影响船舶航向控制和横向移动能力。

#### 3. 推力与角度指示器

用于直观显示推进器当前推力方向和转向状态。

该区域由以下部分组成：

##### 推力输出指示

垂直刻度条用于显示当前推进器推力输出比例。

特点：

- 推力越大，指示位置越高
- 推力越小，指示位置越低
- 实时反映当前动力输出状态

##### 角度指示器

扇形指示区域用于显示推进器当前转向方向。

特点：

- 中间位置表示推进器处于正前方
- 指示区域向左偏移表示左转
- 指示区域向右偏移表示右转
- 偏转幅度与实际推进器角度同步变化

船员可通过该图形快速判断推进器当前朝向，而无需关注具体角度数值。

#### 4. 推进器状态（Status）

显示推进系统当前工作状态。

常见状态包括：

| 状态名称           | 说明                                                       | 建议处理措施                                           |
| ------------------ | ---------------------------------------------------------- | ------------------------------------------------------ |
| 正常               | 推进器运行正常，各项参数处于允许范围内。                   | 无需处理。                                             |
| 堵转               | 推进器受到异物缠绕、卡阻或负载过大，导致电机无法正常转动。 | 检查螺旋桨及推进器周围是否存在障碍物，排除后重新启动。 |
| 电机过温           | 推进电机温度超过安全工作范围。                             | 降低负载或停止运行，待温度恢复正常后再继续使用。       |
| 驱动器过温         | 电机驱动模块（MOS）温度过高。                              | 检查散热系统是否正常工作，必要时暂停运行。             |
| 过流               | 推进器工作电流超过允许范围。                               | 检查推进器负载、螺旋桨状态及供电系统。                 |
| 通信故障           | 无法与推进器控制器正常通信。                               | 检查控制器供电、通信线路及网络连接状态。               |
| 电机温度传感器异常 | 无法正确获取电机温度信息。                                 | 检查温度传感器及相关线路连接。                         |
| 驱动器温度报警     | 驱动器温度接近保护阈值。                                   | 建议降低推进器输出功率，并检查散热情况。               |
| 过压               | 输入电压超过设备允许范围。                                 | 检查电池组、电源模块及充电设备。                       |
| 欠压               | 输入电压低于设备正常工作范围。                             | 检查电池剩余电量并及时充电。                           |
| 电路故障           | 检测到驱动器内部或外部电路异常。                           | 停止使用设备，并联系维护人员进行检查                   |
| 散热风扇故障       | 驱动器散热风扇工作异常。                                   | 检查风扇供电、连接线路及风扇本体。                     |

#### 状态显示规则

#### 使用说明

- 推进器转速、角度和状态均为实时刷新数据。
- 自动航行模式下，推进器角度和转速由控制系统自动调整。
- 手动模式下，推进器响应驾驶员操作指令。

#### 注意事项

- 推进器状态显示“故障”时，应立即停止相关操作并检查设备。
- 推进器信息仅作为监控参考，实际控制效果受船体载荷、水流、风力等环境因素影响。







## 三、船体姿势角度



![船体姿势角度](./HomePage/feature/img/sailingState/船体姿势角度.png)



船舶姿态信息位于主页中部区域，用于实时显示船舶当前航向及姿态状态，包括航向角（Heading）、横滚角（Roll）和俯仰角（Pitch）。

通过姿态信息，船员可及时掌握船舶运动状态和船体姿态变化情况，为航行控制和作业安全提供参考。

#### 1. 航向角（Heading）

航向角表示船首当前指向方向。

单位：

```
°（Degree，度）
```

#### 2. 横滚角（Roll）

横滚角表示船体绕纵向轴（船首至船尾方向）的倾斜角度。

单位：

```
°（Degree，度）
```

横滚角反映船体左右倾斜情况。



#### 3. 俯仰角（Pitch）

俯仰角表示船体绕横向轴（左舷至右舷方向）的倾斜角度。

单位：

```
°（Degree，度）
```

俯仰角反映船首和船尾的上下姿态变化。





## 四、电池信息

![电池信息](./HomePage/feature/img/sailingState/电池信息.png)

电池信息卡片位于主页右上区域，用于实时显示船舶当前位置、电池状态以及供电信息，帮助船员及时掌握船舶能源状况和剩余续航能力。

#### 1. 船体位置纬度（Latitude）

显示船舶当前位置纬度信息。

#### 2. 船体位置经度（Longitude）

显示船舶当前位置经度信息。

#### 3. 电池电量图标

用于图形化显示当前电池剩余电量。

电量图标会根据剩余电量自动变化，帮助船员快速判断电池状态。

#### 4. 电池状态（Status）

显示电池系统当前运行状态。

#### 5. 电池当前电压（Voltage）

显示电池组实时输出电压。

单位：

```
V（伏特）
```

#### 6. 电池当前剩余容量（SOC）

SOC（State of Charge）表示电池剩余可用容量百分比。

单位：

```
%（百分比）
```

# 五、灯光-空调控制



## 一、灯关控制

![灯光控制](./HomePage/feature/img/lightsAirControl/灯光控制.png)

#### 1. 所有灯光控制开关

用于统一控制船舶全部灯光设备。

开启后：

- 导航灯
- 锚灯
- 顶灯
- 地灯

将根据当前配置统一开启。

关闭后：

- 所有受控灯光将统一关闭。

该功能适用于快速开启或关闭整船照明系统。



#### 2. 导航灯控制开关

用于控制船舶导航灯。

#### 3. 锚灯控制开关

用于控制船舶锚灯。

#### 4. 顶灯控制开关

用于控制船体顶部照明灯。



#### 5. 地灯控制开关

用于控制船体地灯或甲板辅助照明灯。



#### 开关状态说明

灯光控制采用滑动开关方式进行操作。

| 开关状态 | 说明           |
| -------- | -------------- |
| 开启     | 对应灯光已打开 |
| 关闭     | 对应灯光已关闭 |

点击或滑动开关即可切换灯光状态。



#### 状态同步机制

灯光控制页面显示的是船体灯光的实际工作状态。

当出现以下情况时，界面状态将自动同步更新：

- 用户在当前页面操作灯光开关
- 用户在智能设备控制页面操作灯光
- 船载控制系统直接控制灯光
- 灯光设备状态发生变化

因此，界面显示状态始终与船体实际灯光状态保持一致。



#### 智能设备控制页面

除主页快捷控制外，系统还在《[智能设备控制页](../../ShipPage/shipPage.md)》中提供完整的灯光管理功能。

智能设备控制页面可显示：

- 每个灯光设备的名称
- 灯光实时状态
- 灯光安装位置
- 灯光控制功能

便于船员快速识别和管理船体各位置灯光设备。

#### 注意事项

- 夜间航行时应按照当地航行规则正确使用导航灯。
- 锚泊期间应根据实际情况开启锚灯。
- 长时间开启高功率照明设备可能增加电池能耗。
- 若开关状态与实际灯光状态不一致，请检查灯光设备和通信状态是否正常。





## 二、空调控制

![空调控制](./HomePage/feature/img/lightsAirControl/空调控制.png)

空调控制区域位于主页右下区域，用于显示和控制船舱空调设备运行状态。

船员可通过该区域快速查看当前空调工作模式、设定温度及风速等级，并进行快捷调节，无需进入智能设备控制页面即可完成常用操作。



#### 1. 空调开关

用于控制空调设备的启动和关闭。

状态说明：

| 状态 | 说明           |
| ---- | -------------- |
| 开启 | 空调正在运行   |
| 关闭 | 空调已停止运行 |

开启后，空调将按照当前设定的模式、温度和风速运行。

#### 2. 空调模式

显示当前空调运行模式。

支持以下模式：

| 模式 | 说明         |
| ---- | ------------ |
| 制冷 | 降低舱内温度 |
| 制热 | 提高舱内温度 |
| 除湿 | 降低空气湿度 |

#### 3. 空调温度

显示当前设定温度。

#### 4. 空调风速

显示当前风速等级。

支持以下档位：

| 风速 | 说明                     |
| ---- | ------------------------ |
| 低速 | 风量较小，运行噪声较低   |

```

#### 7. 空调风速切换按钮

用于切换风速等级。

```
低速 → 中速 → 高速 → 自动
```

每点击一次：

温度降低 1℃
```

#### 9. 升高温度按钮（＋）

用于提高空调设定温度。

每点击一次：

```
温度升高 1℃
```

#### 10. 最高支持温度

显示空调允许设置的最高温度。





#### 注意事项

- 空调关闭状态下无法调整温度和风速。
- 制冷模式下温度设置过低会增加电池能耗。
- 长时间使用高速风速可能增加设备功耗。
- 自动模式下风速由空调设备自动控制。
- 若空调状态长时间未更新，请检查设备连接状态是否正常。

# 十六、语音助手设置

App 提供语音助手功能开关，用于通过语音方式控制船舶智能设备，提升操作便捷性与交互效率。

## 1、语音助手开关

用户可在设置页面开启或关闭语音助手功能。

![语音助手开关](./SettingPage/feature/img/languageAssistant/语音助手开关.jpg)

## 2、权限授权

首次开启语音助手时，系统需要申请麦克风权限。

用户需授权麦克风访问权限，以便语音助手（“小韵”）能够接收语音指令。

![语音权限授予](./SettingPage/feature/img/languageAssistant/语音权限授予.jpg)

## 3、语音唤醒与控制

完成授权后，用户可通过语音唤醒语音助手。

在平板端说出：

> “你好小韵”

即可唤醒语音助手“小韵”，并通过语音指令控制船舶相关智能设备。

用户可在[语音助手](../../XiaoYun/xiaoYun.md)说明页面查看更多功能详情。

![小韵语音助手](./SettingPage/feature/img/languageAssistant/小韵语音助手.jpg)

## 4、说明

- 语音功能依赖麦克风权限，请确保已正确授权
- 语音唤醒词为“你好小韵”
- 语音助手可用于控制灯光、空调等船载设备
- 功能可在设置中随时开启或关闭

# 十七、小韵语音助手

小韵语音助手是一种面向船舶智能设备的语音交互功能，用户无需手动操作平板，即可通过语音指令完成设备控制，实现更高效、便捷的操作体验。

在设置页面[打开语音助手](../SettingPage/feature/languageAssistant.md)功能后，用户可通过唤醒词：

> “你好小韵”

唤醒语音助手并开始语音控制。

![小韵语音助手](./XiaoYun/img/xiaoYun/小韵语音助手.jpg)

## 1、核心能力

小韵语音助手支持对船舶智能设备的多维度控制，主要包括以下能力：

- 灯光系统控制（整体或单设备开关）
- 空调系统控制（开关、模式、温度、风速等）
- 航行辅助功能控制（如自动驾驶模式开关）

通过自然语言指令，用户可以实现对设备的快速操作，无需进入具体控制页面。

## 2、支持的语音指令范围

小韵语音助手支持多种自然语言表达方式，系统会自动识别用户意图并映射到对应控制行为。

### 1. 灯光控制

支持对全船灯光或单个灯具进行开关控制，例如顶灯、导航灯、地灯、锚灯等均可独立控制。

同时支持“全部灯光开/关”等批量控制指令。

------

### 2. 空调控制

支持空调系统的完整控制能力，包括：

- 开关控制
- 温度调节（精确到具体温度或模糊表达，如升高/降低一点）
- 模式切换（制冷 / 制热 / 除湿 / 通风 / 自动等）
- 风速调节（高、中、低、自动档位）
- 扫风控制

同时支持自然语言表达，例如“温度高一点”“风速调低”等模糊指令。

------

### 3. 自动驾驶控制

支持对船舶自动驾驶模式的开启与关闭控制，并兼容多种表达方式（如“开启自动驾驶”“打开自动模式”等）。





## 3、说明

- 使用语音功能前需确保已开启语音助手并授予麦克风权限
- 唤醒词为“你好小韵”
- 语音指令会实时映射到对应设备控制逻辑
- 建议在环境噪音较低时使用以获得更高识别准确率

# 十八、相关术语

**循迹：**指船舶按照预设航线自动航行的功能。

**航线：**由多个路径点按顺序连接形成的航行路径，可用于循迹任务。

**路径点：**航线中的基础节点，用于定义船舶航行路径的关键位置。

**电子围栏：**在地图上定义的安全区域边界，用于限制船舶航行范围。

**越界报警：**当船舶驶出电子围栏区域时触发的声光提示机制。

**靠泊：**指船舶靠近并对准泊位区域的操作过程。

**泊位：**用于船舶停靠的目标区域，在系统中以矩形区域表示。
