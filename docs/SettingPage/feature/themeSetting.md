# 主题切换

为适配不同光照环境（如强光与弱光场景），提升界面可视性与使用舒适度，App 提供主题切换功能。

用户可通过点击切换按钮，在不同主题模式之间快速切换，以获得更合适的视觉体验。

![主题切换](./img/themeSetting/主题切换.jpg)

## 深色模式效果：

深色模式适用于弱光或夜间环境，可有效降低屏幕亮度刺激，提升长时间使用的舒适性。在深色模式下，地图与界面整体采用深色风格显示，以增强对比度并突出关键操作元素。
