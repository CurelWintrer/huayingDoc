# 船舶设置

船舶设置页面用于配置船舶运行相关的核心参数，包括航速限制、船体尺寸、电池参数以及推进器性能等，以确保系统显示与控制逻辑与实际船舶状态一致。

![设置首页](../img/settingPage/设置首页.jpg)

1. **最大 / 最小航速设置**

   用于设置船舶允许的最大与最小航行速度。

   - 最大航速用于仪表盘速度范围显示
   - 同时用于速度滑块上限限制，避免误操作导致过高速度输入，提升航行安全性

2. **船体尺寸设置**
    用于配置船体实际尺寸参数。
    在靠泊页面中，泊位选择框会根据船体尺寸进行地图比例映射，从而提高泊位选择的精确度与实际贴合度。

3. **电池设置**
    用于配置电池参数与电量显示方式，包括：

   - 电池最大电压
   - 电池最小电压
   - 电量显示模式

   当电量显示模式设置为“电池电压”时，系统将根据当前电压值计算并显示剩余电量。

4. **推进器设置**

   用于配置推进系统的性能参数，包括：

   - 最大转速限制
   - 最大角度范围

## 说明

- 所有参数用于统一系统显示与真实设备状态
- 参数设置将直接影响仪表盘显示与控制逻辑
- 建议根据实际船舶硬件参数进行配置，避免偏差
