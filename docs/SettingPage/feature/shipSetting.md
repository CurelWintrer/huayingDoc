# 船舶设置



![设置首页](../img/settingPage/设置首页.jpg)

船舶设置页面主要包含以下功能：

1. 最大、最小航速设置：最大航速主要用于速递仪表盘显示最大值，还有速度设置滑块，限制其最大值，避免误操作发送出太大的速度，导致安全隐患。
2. 船体大小设置：在靠泊界面的泊位选择框是按照实际的船体大小在地图上进行了映射，正确设置船体大小将帮助您更加精确地选择泊位。
3. 电池设置：这里可以调整电池的最大和最小电压，以及电池电量显示模式，当电量显示模式为“电池电压”时，App将通过当前船舶电池的电压计算剩余电量。
4. 推进器设置：这里可以调整推进器的最大转速以及最大角度。
