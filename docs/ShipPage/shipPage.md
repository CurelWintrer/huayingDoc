# 智能设备控制页面

该页面用于对船载智能设备进行集中控制，主要包括灯光与空调系统的控制功能。用户可在此页面实时查看设备状态，并进行调整。



![灯光空调控制页面](./img/shipPage/灯光空调控制页面.png)

1. **灯光控制开关：**用于开启或关闭船舶灯光系统
2. **灯光状态指示：**显示当前灯光状态
   - 开启状态：橙色显示
   - 关闭状态：白色显示
3. **空调开关：**用于开启或关闭空调系统
4. **空调当前设定温度：**显示当前空调目标温度
5. **空调当前模式：**显示当前运行模式（如制冷 / 制热 / 除湿等）
6. **空调当前风速：**显示当前风速档位
7. **空调温度调低按钮：**每次点击降低设定温度
8. **空调温度调高按钮：**每次点击提高设定温度
9. **空调模式切换按钮：**用于切换空调运行模式
10. **空调风速调节按钮：**用于调整空调风速档位
